Bộ Skills Tối Ưu Cho Quá Trình Vibe Coding Dự Án "Focus-Node"
Dựa trên định hướng sản phẩm (rule.txt) và các yêu cầu kỹ thuật của một dự án Java/JavaFX Desktop App, dưới đây là hệ thống các Skill (Kỹ năng đặc tả) mạnh mẽ nhất mà bạn nên sử dụng kết hợp (bằng cách dùng lệnh /tên-skill hoặc nhắc tôi gọi chúng) trong từng giai đoạn phát triển.

User Review Required
Bạn hãy xem xét danh sách này. Nếu đồng ý với hệ thống skill này, chúng ta sẽ áp dụng chúng như một "bộ công cụ tiêu chuẩn" cho toàn bộ quá trình phát triển các tính năng tiếp theo (Knowledge Base, Tasks, Pomodoro, LAN Hub).

1. Nhóm Kỹ Năng: Lập Kế Hoạch & Kiến Trúc Sản Phẩm
Trước khi gõ bất kỳ dòng code nào, cần định hình cấu trúc và kế hoạch rõ ràng để tránh "nợ kỹ thuật" (technical debt).

/plan-writing: Kỹ năng cốt lõi bắt buộc. Sử dụng để bóc tách yêu cầu phức tạp thành các task nhỏ, viết file implementation_plan.md rõ ràng trước khi code.
/senior-architect: Cung cấp tư duy kiến trúc sư, giúp thiết kế mô hình MVC chuẩn chỉ, luồng dữ liệu (Data Flow) rõ ràng cho toàn hệ thống.
/product-manager-toolkit: Giúp bám sát rule.txt, đảm bảo tính năng được làm ra thực sự giải quyết nỗi đau của sinh viên/người học.
2. Nhóm Kỹ Năng: Lập Trình Java & Xử Lý Logic (Core)
Phát triển logic nền tảng, quản lý đa luồng (Threads cho Pomodoro và LAN).

/java-pro: Master Java. Cung cấp các best practices, clean code, và quản lý bộ nhớ tốt cho ứng dụng Java.
/backend-architect: Tối ưu hóa cấu trúc luồng xử lý dữ liệu ẩn dưới UI (Service Layer, Repository Layer).
/clean-code: (Kỹ năng ẩn) Giúp giữ cho các Class controller không bị phình to, chia nhỏ các hàm hợp lý.
3. Nhóm Kỹ Năng: Thiết Kế UI/UX (Frontend & JavaFX)
Tạo ra giao diện "Vibe", hiện đại, mượt mà và không giống các app Java khô khan truyền thống.

/ui-pattern: Chuyên tạo các pattern tái sử dụng (như NoteCard, TaskItem, Dialog). Giúp code FXML gọn gàng và đồng bộ.
/frontend-design: Kỹ năng thiết kế frontend. Đảm bảo quy tắc về Typography, màu sắc, khoảng cách (Spacing & Padding) luôn chuẩn mực.
/design-spells: Thêm các "ma thuật" vào UI như hiệu ứng hover mềm mại, đổ bóng (drop shadow), bo góc chuẩn (border-radius), giúp app trông cao cấp.
/ux-copy: Kỹ năng viết text UI (Microcopy). Giúp các thông báo lỗi, placeholder (vd: "Write something...") trở nên thân thiện và động viên người học.
4. Nhóm Kỹ Năng: Cơ Sở Dữ Liệu (Database - SQLite)
Quản lý nơi lưu trữ ghi chú, task và lịch sử học tập bền vững.

/database-design: Giúp thiết kế Schema tối ưu cho Note, Task, Focus Session, Tags có liên kết với nhau (Relational DB).
/sql-pro: Tối ưu hóa các câu lệnh truy vấn, đảm bảo khi người dùng có hàng ngàn note thì app vẫn search mượt mà.
5. Nhóm Kỹ Năng: Xử Lý Mạng (LAN Hub)
Dành riêng cho tính năng độc đáo nhất: Học nhóm qua mạng LAN.

/network-engineer: Chuyên gia về mạng. Giúp xử lý các kết nối TCP/UDP bằng Java Socket một cách ổn định, quản lý rớt mạng, đồng bộ trạng thái giữa các máy.
/concurrency-patterns: (Hoặc tương đương trong java-pro) Xử lý tranh chấp luồng khi nhiều người cùng gửi dữ liệu trong phòng LAN.
6. Nhóm Kỹ Năng: Debug & Đảm Bảo Chất Lượng
Khi Vibe Coding sinh ra code nhanh, rất cần hệ thống kiểm tra để không sập app.

/vibe-code-auditor: Kiểm tra các đoạn code được AI sinh ra quá nhanh xem có lỗ hổng cấu trúc hoặc dễ vỡ (fragile) hay không.
/systematic-debugging: Phương pháp debug có hệ thống. Khi gặp lỗi JavaFX (vd: Not on FX Application Thread), skill này giúp khoanh vùng và xử lý triệt để thay vì sửa mò.
/code-reviewer: Đóng vai trò như một senior review lại PR, đảm bảo code mới không phá hỏng các tính năng cũ.
💡 Cách Sử Dụng Khuyến Nghị (Workflow)
Mỗi khi bạn giao một task mới (ví dụ: "Làm chức năng Pomodoro"), hãy gọi: "Tạo tính năng Pomodoro nhé /plan-writing /java-pro /ui-pattern"

Hệ thống sẽ tự động kích hoạt tư duy lập kế hoạch -> viết code Java xịn -> làm UI đẹp.
